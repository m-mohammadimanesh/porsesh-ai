## Description | توضیحات

**English:**
Porsesh AI is an intelligent chatbot application built with Next.js for the frontend and FastAPI for the backend. It leverages Google's Gemini AI to provide insightful and context-aware responses, and utilizes ChromaDB for vector storage and PyPDF2 for document parsing.

**فارسی:**
پرسش یک اپلیکیشن هوشمند چت‌بات است که بخش فرانت‌اند آن با Next.js و بک‌اند آن با FastAPI ساخته شده است. این پروژه از هوش مصنوعی Gemini گوگل برای ارائه پاسخ‌های دقیق و آگاه از متن استفاده می‌کند و از ChromaDB برای ذخیره بردارها و PyPDF2 برای پردازش اسناد بهره می‌برد.

## Features | امکانات

- 🤖 AI-powered chat using Google Gemini
- 📄 PDF document parsing and analysis
- 🧠 Vector search with ChromaDB
- ⚡ Fast and responsive UI with Next.js & Tailwind CSS
- 🐍 High-performance backend with FastAPI

- 🤖 چت مبتنی بر هوش مصنوعی با استفاده از Google Gemini
- 📄 پردازش و تحلیل اسناد PDF
- 🧠 جستجوی برداری با ChromaDB
- ⚡ رابط کاربری سریع و واکنش‌گرا با Next.js و Tailwind CSS
- 🐍 بک‌اند با کارایی بالا با FastAPI
